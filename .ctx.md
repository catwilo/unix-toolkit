## Decisiones
- Symlinks: repo es fuente de verdad; instalador limpia y revincula
- Plataformas: termux | debian | raspbian | mac (escalable)
- configs/: cada plataforma hace source de shared + agrega lo suyo
- Mismo nombre en shared y plataforma = error, script aborta
- nvim lite: config sin plugins, instalables uno a uno con uninstall seguro
- nvim full: config actual completa
- Opcionales (mpd, byobu, repair-nix): no corren por defecto
- Raspbian = rama propia ARM 32-bit, no alias de debian
- Backup .bak si destino existe al crear symlink
