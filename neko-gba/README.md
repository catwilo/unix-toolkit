# neko-gba

Scaffolds a complete Game Boy Advance emulator project under `~/neko-gba` in one step — a Rust workspace (Cargo) organized into core modules for GBA, audio, and video, with dependencies pinned for reproducibility. Starting a new project this way gives a consistent layout instead of assembling it by hand.

## Usage

```sh
./neko-gba-setup.sh      # creates the project tree at ~/neko-gba
```

Creates directories, stub source files (with package/module declarations), and final configuration files.
